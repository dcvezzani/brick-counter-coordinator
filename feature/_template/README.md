Copy this folder to `feature/<kebab-slug>/` when starting a new AIDLC Feature.

Canonical spec templates also live in the AI-DLC submodule:

- [product-spec-template.md](../../.claude/deps/ai-dlc/skills/spec-management/templates/product-spec-template.md)
- [tech-spec-template.md](../../.claude/deps/ai-dlc/skills/spec-management/templates/tech-spec-template.md)

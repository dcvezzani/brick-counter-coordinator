The existing query parameter values are fine to use as defaults; only these will need to change
- strCatType=P; for the present, we will only focus on PARTS; we might support other types like MINIFIGS in the future
- strInvNew=U; "U": used, "N": new
- strQuery=3001; targeted part number

When querying for a specific store inventory lot, these query parameters are important:
- n4InvID=536108997; the store inventory lot id
- strQuery=; (empty value)
